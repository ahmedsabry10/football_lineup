# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-15

### Added

- Initial release of `football_lineup`.
- Support for 11-a-side, 7-a-side, and 5-a-side formations.
- `DynamicLineupScreen` with configurable field type, colors, and gradient.
- Team selector (tabs) with optional custom builder and style parameters.
- Manager/coach and player count (e.g. 11/11) info bar with customizable colors and visibility.
- Drag-and-drop position swapping with optional callback.
- Player tap callback and formation change callback.
- Multiple name label styles (compact, minimal, badge, gradient) for player avatars.
- Single-team mode support.
